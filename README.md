# Mobiroller_HomeProject

This project has developed for **Mobiroller** as a test project.

## Modules:
  ### Data
    - Model; Contains data models.
    - Service; Contains service modules, such as CRUD methods and repository for lifecycle.
  ### Utils
    - Contains methods that has basic algorithms such as getting current date.
  ### View
    - Activity; Contains activity classes.
    - Adapter; Contains adapters for recycler views.
    - Alert_Dialog; Contains alert dialog modules that reachable from out of activities.
  ### View_Model
    - Contains view models for user interfaces.

## Model:
* Arhcitecture: MVVM

## Dependencies:

* Lifecycle; ``` implementation 'androidx.lifecycle:lifecycle-extensions:2.2.0' ```
* Firebase Realtime Database; ```implementation 'com.google.firebase:firebase-database:19.6.0'```
* Firebase Storage; ``` implementation 'com.google.firebase:firebase-storage:19.2.1' ```	
* Glide; ``` implementation 'com.github.bumptech.glide:glide:4.12.0' ``` ``` annotationProcessor 'com.github.bumptech.glide:compiler:4.12.0' ```
